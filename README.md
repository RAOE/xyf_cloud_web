#### 安装教程
0.  maven install course(root) 
1.  启动eureka
2.  启动gateway
3.  启动service
4.  启动system
5.  前端 cd admin 
6.  npm install 
7.  npm run serve 
8.  启动完毕后：http://localhost:8080 进入首页
